## A17 – Discover 10 different types of locks in use


One of the most common locks I noticed is the padlock. I found this one very practical because it can be used on many different things such as gates, chains, lockers, storage boxes, and even bags. What makes it different from most door locks is that it is not fixed inside the door itself. It is a separate lock that can be attached and removed whenever needed. I found it interesting because it is such a simple lock, but it is still used everywhere in everyday life. It works by securing a metal shackle in place, and depending on the type, it can be opened either with a key or with a number combination.

Another common lock is the deadbolt lock, which is often used on house doors. I found this one important because it is much more security-focused than an ordinary spring lock. A deadbolt works by sliding a solid metal bolt deeply into the door frame, which makes it harder to force the door open. This is why it is so commonly used as the main lock on home entrances. It stood out to me because many people may think the handle lock is already enough, but the deadbolt is really the part that adds stronger protection.

A knob lock is another lock that is used very commonly, especially on indoor doors and sometimes on main doors together with another lock. I noticed that this type of lock is built directly into the round door knob, which makes it simple and quick to use. However, after learning more about it, I understood that it is usually not the strongest lock on its own because the locking part is connected to the handle area, which can be weaker under force. That makes it a useful everyday lock, but often better when paired with something stronger like a deadbolt.

A lever handle lock is similar to a knob lock, but instead of a round knob it uses a lever handle. I found this type interesting because it is very common in offices, schools, and other buildings where people need easy and quick access. It is often more comfortable to use than a knob, especially for children, elderly people, or anyone carrying bags or books. In terms of security, it can work well for normal rooms and controlled indoor spaces, but like a knob lock, it is not always the strongest option by itself for a main entrance. What makes it different is more in the way it is designed and used.

A remote-unlock intercom lock is a digital lock connected to an intercom or doorbell system where someone inside can unlock the door electronically. I found this useful because it combines communication with access control. Instead of giving every visitor a key or code, the person inside can first speak to them or check who they are and then unlock the door remotely. This is common in apartments, offices, and gated buildings, and it makes the lock part of a wider entry-management system rather than only a simple piece of door hardware.

One digital lock I found very common is the PIN code door lock. This is used on doors where, instead of using a key, the user enters a number code on a keypad. I found this interesting because it changes access from something physical, like carrying a key, into something based on remembering the correct code. It is very practical in homes, offices, and shared spaces because the code can be changed more easily than replacing a whole lock. It also makes entry more convenient for people who do not want to carry keys all the time.

A fingerprint door lock is another digital lock that stood out to me. This type of lock uses biometric identity, which means the door opens only if the stored fingerprint matches the person trying to enter. I found this interesting because it feels much more personal than using a key or code. The lock is checking something physical about the person rather than an object they carry or a number they remember. This makes it a strong example of how locking systems are becoming more closely linked to identity and are moving beyond traditional keys.

An RFID card lock is a digital lock that opens when a user taps a card, tag, or key fob on a reader. I found this one very common in hotels, offices, and staff-only areas. What makes it different from a normal key is that the lock checks a stored digital credential instead of the physical shape of a key. ASSA ABLOY explains that RFID, which stands for radio frequency identification, works through communication between an RFID tag and a reader using electromagnetic fields. In simple terms, when the card is brought close to the lock, the reader checks the information stored in that card and decides whether access should be granted. I found this to be a strong example of a digital lock because it makes access faster to manage in places with many users, and it also shows how entry systems are becoming more electronic and identity-based rather than mechanical.

A face recognition lock is a more advanced digital lock that checks facial features before opening. I found this one especially interesting because it feels very modern and contactless. Instead of touching the lock or typing anything, the system scans the face and decides whether access should be allowed. This makes it useful in places where quick and hands-free access matters. It is a strong example of how digital locks are moving away from mechanical entry and toward automated identity-based entry.

A voice-controlled smart lock is one of the more unusual digital locks I found. What makes it interesting is that it does not work like a normal key or keypad lock. Instead, it connects to a smart-home system and can respond to voice commands through a voice assistant. Yale’s support information explains that its smart locks can work with Google Home and that a user can say commands such as “OK Google, lock my front door” or ask if the door is locked. For unlocking, the system asks for a security PIN, which means the lock is not opened by voice alone. I found this important because it shows that even when a lock becomes more convenient and modern, there is still an extra security step built in. That makes it a strong example of a digital lock because it combines automation, remote access, and voice control while still using a second check to reduce misuse.
