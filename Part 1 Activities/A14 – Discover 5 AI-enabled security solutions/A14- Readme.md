# A14 – Discover 5 AI-enabled security solutions


One AI-enabled security solution I found very interesting is Microsoft Security Copilot. I came across it while looking at how large cybersecurity companies are now using AI in real security work rather than only in chat-style tools. Microsoft describes it as a generative AI-powered security solution that helps defenders respond to threats at machine speed and scale, and it can work with products such as Microsoft Defender, Sentinel, Intune, and Entra. What stood out to me is that it is not just giving basic answers like a normal chatbot. It is designed to help security teams investigate incidents, understand alerts, gather threat intelligence, and even automate parts of their workflow. That makes it a strong AI-enabled security solution because the AI is being used directly inside real defensive work instead of only as a general productivity tool.

One solution that stood out to me is Gemini in Google Security Operations (SecOps). I found this especially strong because it shows AI being used inside a major cloud security platform rather than as a separate tool. Google explains that Gemini in Security Operations can help analysts query security data in natural language, create detection rules, build playbooks, investigate threats, and surface recommended actions. I think this makes it a strong AI-enabled security solution because it helps security teams interact with very complex security data more easily and move faster from investigation to action. It also shows that AI in cybersecurity is not only about detection, but also about helping teams understand and use their security platforms better.

A solution that stood out to me from IBM is QRadar Investigation Assistant. IBM describes it as an AI-powered application that uses watsonx.ai to streamline investigations, enhance productivity, and help analysts uncover threats more efficiently. IBM’s documentation also explains that it can generate offense summaries that help analysts quickly understand attack vectors, affected users, and impacted assets. I found this to be a strong AI-enabled security solution because investigation is one of the most time-consuming parts of security work, and this tool is clearly designed to reduce that effort. It fits this task well because it shows AI being used to help analysts make sense of complex security incidents instead of starting every investigation from scratch.

One AI-enabled security solution I found quite interesting is Proofpoint’s NexusAI, because it focuses on email threats rather than general security operations. I came across it while looking at how companies defend people from phishing, business email compromise, and malicious attachments. Proofpoint says NexusAI uses advanced machine learning to classify phishing emails, malware, spam, and BEC threats. What makes this solution feel different from many other AI tools is that it is centred on human-targeted attacks through email, which is still one of the most common ways people get exploited online. I found it strong because it shows AI being used to filter and understand suspicious communication before it reaches the user, rather than only helping analysts after the attack has already started.

Another AI-enabled security solution I found is the Menlo Browser Security Platform. This one stood out because it is focused on the browser itself, which is where people do most of their online work. Menlo explains that its platform secures both human users and AI agents through a single browser-security platform, and its materials also mention threat protection, secure enterprise browsing, data protection, and file security. I found this different from email or endpoint tools because it protects the place where users actually open websites, click links, download files, and interact with web apps. That makes it a strong AI-enabled security solution because it shows AI being used to secure browsing activity directly, which is a very real and practical attack surface.
